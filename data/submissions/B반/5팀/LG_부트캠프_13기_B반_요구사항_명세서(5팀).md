# 요구사항 명세서

> **프로젝트명:** WaveCare — 욕실 안전 가드 (레이더 기반 제스처 인식 + 낙상 감지)  
> **반/팀:** B반 5팀  
> **작성일:** 2026-07-01  
> **버전:** 1.0

---

## 프로젝트 개요

인피니언 mmWave 레이더 센서(1Hz, target-level)를 단독으로 사용하여, 욕실 환경에서 **제스처 인식**(3종)과 **낙상 감지**를 수행하는 온디바이스 안전 시스템.

카메라를 설치할 수 없는 프라이버시 민감 공간(욕실)에서, 레이더의 비영상 데이터(Range, Azimuth, Speed, Elevation, Doppler)만으로 사용자의 제스처와 위험 상태를 인지한다.

| R-06 | Air Control 동작 | AWAKE 상태에서 박수 제스처 인식 시 가상 기기의 바람 세기가 변동됨 | 00:12 |
| R-07 | X 그리기 제스처 인식 | 손으로 크게 X를 그리면 해당 제스처로 분류됨. Confidence 60% 이상일 때 감지 | 00:17 |

| R-09 | 낙상 감지 | 사용자의 Elevation이 임계값 미만으로 떨어지고 장시간 미동(무움직임)이 지속될 때 "낙상" 판정 | 00:30 |

| R-12 | 가상 기기 상태 + 설명가능 로그 출력 | 명령마다 입력 신호 값·신뢰도·결정 근거가 터미널/GUI에 함께 출력됨 | 00:00 |
| R-13 | 처리 실시간성 | 1Hz 입력에 대해 다음 프레임 도착 전에 처리가 완료됨 | N/A (영상 미시연 기능) |

## 비고

- 요구사항 ID는 기능/비기능 구분 없이 중립 번호(`R-01`, `R-02`, …)를 사용합니다.
- 영상으로 시연할 수 없는 요구사항(R-01, R-11, R-13, R-14)은 시연 타임스탬프에 `N/A (사유)`로 표기합니다.
- 시연 영상 녹화 완료 후 각 요구사항의 동작 장면에 해당하는 타임스탬프를 기입합니다.
- 검증 기준은 테스트 코드(`tests/`)와 직접 연결됩니다.
