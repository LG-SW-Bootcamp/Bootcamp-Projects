프로젝트 개요

Dadumi는 백그라운드에서 네이티브로 실행되는 크로스 플랫폼(Windows, macOS, Linux) 데스크탑 글쓰기 어시스턴트입니다. 사용자가 텍스트 입력이 가능한 모든 환경(Slack, Notion, Chrome, MS Word, 메모장 등)에서 전역 단축키로 AI를 직접 호출하고, 현재 선택한 텍스트를 AI로 처리한 뒤 결과를 자동으로 원래 위치에 붙여넣을 수 있게 합니다. 수동 복사-붙여넣기 과정을 완전히 없애는 것이 핵심 목표입니다.

비고

  • 텍스트 캡처 방식: 접근성 API(macOS AXUIElement / Windows UI Automation)를 우선 사용하며, 실패 시 클립보드 + 키 입력 에뮬레이션(Cmd+C / Ctrl+C)으로 폴백한다.
  • 붙여넣기 방식: 접근성 API를 우선 사용하며, 실패 시 클립보드 + 키 입력 에뮬레이션(Cmd+V / Ctrl+V)으로 폴백한다. 원래 클립보드 내용은 작업 전 임시 저장 후 복원한다.
  • 단축키 중복 방지: 단축키 연속 입력 시 250ms 디바운싱을 적용하여 중복 실행을 방지한다.
  • 제출 형식: 소스코드는 .zip 파일이 아닌 src/ 디렉토리로 직접 제출됩니다. 테스트 코드는 tests/ 디렉토리가 아닌 src/src/ 내에 포함되어 있으며, 이는 tests/README.md에 명시되어 있습니다.
  • 환경 변수 설정: 별도의 .env 파일은 필요하지 않습니다. API 키를 포함한 모든 설정은 앱 내 설정 창(Settings Window)에서 직접 입력 및 관리되도록 설계되었으며, 환경 변수 파일은 의도적으로 사용하지 않습니다. 앱 실행에 필요한 외부 의존 설정 값은 없으며, API 키만 앱 UI를 통해 입력하면 됩니다.
